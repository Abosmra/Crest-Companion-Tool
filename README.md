# Project Discontinued

This repository no longer hosts the original tool.

GitHub reviewed the repository and related account activity. The review identified content connected to game cheats and exploits. GitHub Acceptable Use Policies prohibit distribution or support of software used for cheating in online games. As a result, the repository content was removed and the project ended.

Development of the tool stopped and all related content was removed.

The repository now serves only as a notice explaining the shutdown.

The project remains discontinued.
